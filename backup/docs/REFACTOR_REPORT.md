## 🚀 核心改进

### 1. 统一配置文件 (`vimrc`)
**解决的问题**:
- ✅ 消除了 .vimrc (668行) 和 macvim_config.vim (236行) 的重复设置
- ✅ 智能检测环境 (vim/nvim/macvim, terminal/GUI)
- ✅ 条件加载，避免不兼容设置

**新增特性**:
```vim
" 环境智能检测
let g:is_nvim = has('nvim')
let g:is_gui = has('gui_running') || has('gui_macvim')
let g:is_macvim = has('gui_macvim')

" 条件配置块
if g:is_gui
    " GUI 专用设置
endif

if g:is_macvim
    " MacVim 专用优化
endif
```

### 2. 统一管理脚本 (`vim-manager`)
**合并的原脚本功能**:
- ✅ `deploy.sh` - 一键部署
- ✅ `install_plugins.sh` - 插件安装
- ✅ `manage_plugins.sh` - 插件管理
- ✅ `setup_vim.sh` - 环境配置
- ✅ `start_macvim.sh` - MacVim 启动
- ✅ `link_vimrc.sh` - 配置链接

**统一命令接口**:
```bash
./vim-manager install          # 完整安装
./vim-manager plugins list     # 插件管理
./vim-manager start            # 启动 MacVim
./vim-manager status           # 状态检查
```
