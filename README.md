- 📁 强大的文件浏览和搜索功能
- 🛠️ 完整的开发工具链支持
- 🔄 简单的更新和维护

## 快速开始

```bash
# 克隆仓库
git clone <your-repo-url> ~/vim_settings

# 进入目录
cd ~/vim_settings

# 运行安装脚本
./vim-manager install
```
