## v3.3 - 2026-03-13 - 功能完善与自动化增强

### 🚀 新增功能

**文件打开与命令工作流优化**：
- `Ctrl-p`：项目级文件查找（基于 fzf.vim）
- `Space fc`：命令面板
- `Space fb`：缓冲区面板
- `Space fh`：命令历史
- `Space fe`：预填充 `:edit` 命令
- `Space fq`：未保存时确认后关闭窗口
- `Space fQ`：强制退出所有窗口
- `Space fw`：保存当前文件
- `Space fx`：保存并退出
- 移除 `Space f` 绑定，避免冲突

**CSV 文件支持**：
- 引入 `csv.vim` 插件，提供列感知查看和编辑
- 自动对齐 CSV/TSV 文件（文件大小 < 2MB）
- 当前列高亮显示
- 支持文件类型：`*.csv`、`*.dat`、`*.tsv`、`*.tab`
- 常用命令：`:CSVTable`、`:%ArrangeColumn`、`:CSVSort`、`:WhatColumn`

**Markdown 优化**：
- Markdown 文件默认启用软换行和断行
- 支持 `markdown`、`mkd`、`pandoc` 文件类型

**自动化安装与管理**：
- 新增 `./vim-manager deps` 命令：安装外部依赖（`vim`、`rg`、`fzf`、`node`）
- 新增 `./vim-manager bootstrap` 命令：从零开始完整安装（依赖 + 配置 + 插件）
- 新增 `scripts/install_dependencies.sh`：独立依赖安装脚本
- 新增 `scripts/bootstrap.sh`：完整自动化安装脚本
- `install` 和 `update` 命令自动执行插件后处理步骤
- 自动为 `markdown-preview.nvim` 安装 Node 依赖
- 自动刷新 helptags

### 🔧 改进优化

**用户体验优化**：
- 启用 `hidden` 模式：切换缓冲区时未保存文件可在后台保持
- 启用 `confirm` 选项：退出时提供保存/丢弃确认
- 减少 `no write since last change` 提示的打断感
- 撤销目录迁移到仓库外本地路径，避免 Git 污染

**macOS 兼容性**：
- 输入法自动切换逻辑默认关闭，避免终端自动化权限弹窗
- 通过显式变量开关控制输入法切换功能

**状态检查增强**：
- `./vim-manager status` 新增外部工具检查（`git`、`vim`、`rg`、`fzf`、`node`）
- 改进符号链接状态检查逻辑

### 🗑️ 清理与移除

**移除冗余**：
- 删除 `all-the-icons` 插件及配置（仅在 MacVim 中使用）
- 删除 `nvim-web-devicons` 插件（不再使用）
- 移除 Neovim 相关配置和文档
- 删除 Copilot 相关功能与文档
- 清理未托管的插件目录

**文档重组**：
- `docs/` 目录重组为：`reference`、`worknotes`、`plans`、`legacy`、`notes`
- 更新所有文档链接和路径
- 修正 `history/` 相关引用为 `worknotes/`

### 📝 配置结构优化

**插件管理**：
- 清理 `settings/plugins.vim` 中的无效插件引用
- 统一插件清单为当前实际使用的 14 个插件

**配置收敛**：
- 所有 autocmd 使用命名 augroup
- 统一递归/非递归映射（nnoremap、vnoremap、inoremap）
- 移除重复的 NERDTree 行为定义
- markdown-preview 路径改为动态解析

## v3.0 - 2025-06-16 - 模块化重构

### 🎯 重大更新：完全重构与模块化

### 概述

本次更新按照奥卡姆剃刀原则对 `vim-manager` 进行完全重构，在保持所有核心功能的同时，大幅简化代码并提升可维护性。重构后，脚本从 1075 行精简到模块化架构，实现了 63% 的代码减少。

### 🚀 重构成果

**代码优化效果**：
- **代码行数**：从 1075 行减少到 396 行（主文件仅 35 行）
- **模块化设计**：拆分为 4 个专用模块，提升可维护性
- **功能聚焦**：移除冗余功能，专注核心 Vim 配置管理

**新的模块化架构**：
```text
vim-manager (35 行 - 主入口)
vim_manager_modules/
├── config.sh (33 行 - 配置)
├── helpers.sh (12 行 - 工具函数)
├── core_logic.sh (89 行 - 核心逻辑)
└── commands.sh (227 行 - 命令处理器)
```

### ✨ 新功能

**模块化系统**：
- 采用清晰的模块化设计，代码组织更合理
- 主入口文件通过 `source` 导入模块
- 每个模块处理特定功能，便于维护和扩展

**插件管理优化**：
- 更新 `PLUGINS` 数组，完美匹配当前项目结构
- 支持所有 12 个实际存在的插件
- 改进插件状态检查和版本显示

### 🛠️ 改进

**代码质量提升**：
- 移除大量冗余功能（主题下载、环境检测、MacVim 功能等）
- 精简函数结构，提升执行效率
- 改进错误处理和用户反馈

**用户体验改进**：
- 保持原有核心命令接口不变
- 优化输出信息的清晰度
- 提升脚本响应速度

### 🗑️ 移除功能

按照奥卡姆剃刀原则，移除与核心功能无关的以下特性：

**主题管理模块**（9 个函数）：
- 主题下载和安装功能
- 主题切换器生成
- 命令行主题切换界面

**环境检测与安装模块**（8 个函数）：
- 操作系统和 Vim 类型检测
- 自动 Vim/MacVim 安装
- 字体安装功能

**扩展插件管理**（3 个函数）：
- 动态插件添加/删除
- 与核心管理重叠的功能

**其他冗余功能**：
- MacVim 启动器
- 重复的状态检查函数
- 未实现的函数存根

### 🔧 保留的核心功能

所有重要的配置管理功能均保留：
- `install` - 安装配置和插件
- `update` - 更新所有托管插件
- `clean` - 清理未托管插件
- `status` - 检查配置和插件状态
- `uninstall` - 完整配置移除
- `help` - 显示帮助信息

### 📋 升级说明

**自动升级**：
- 重构后的脚本向后兼容
- 所有核心命令保持不变
- 用户无需修改使用习惯

**新用户**：
- 更快的安装体验，代码更简洁

## 附录：完整插件清单

当前版本（v3.3）托管的 14 个插件：

| 插件 | 功能 |
|------|------|
| `vim-airline` | 美观且信息丰富的状态栏 |
| `vim-airline-themes` | 为 airline 提供多种主题 |
| `nerdtree` | 树形文件系统浏览器 |
| `vim-devicons` | 为文件和文件夹提供图标 |
| `vim-fugitive` | 强大的 Git 命令集成 |
| `vim-surround` | 快速添加/修改/删除包围字符 |
| `vim-commentary` | 智能注释/取消注释 |
| `vim-polyglot` | 多种编程语言的语法高亮 |
| `auto-pairs` | 自动匹配括号、引号等 |
| `vim-sensible` | 提供合理的 Vim 默认设置 |
| `fzf.vim` | 基于 fzf 的文件搜索和模糊匹配 |
| `csv.vim` | 改善 CSV 文件的列感知显示与处理 |
| `markdown-preview.nvim` | 在浏览器中预览 Markdown |
| `vim-which-key` | 显示 Leader 键可用分组与命令 |
