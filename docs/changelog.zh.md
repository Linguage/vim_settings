# 更新日志 - `vim-manager` 更新

## 日期: 2025-06-16

### 概述

本次更新为项目引入了一个重要的增强功能：一个名为 `vim-manager` 的专用配置管理脚本。此次更新的主要目标是解决可移植性的挑战，使得这个高度定制化的 Vim 环境能够通过单一命令在任何机器上快速复制。

### 解决方案：`vim-manager`

`vim-manager` 是一个功能强大、自包含的 shell 脚本，它自动化了 Vim 配置的整个生命周期。它作为插件管理和设置的真实单一来源。

#### 新增的核心功能：

-   **插件清单**：现在所有必需的插件都在脚本中明确定义，使得添加、删除或更新依赖项变得简单。

-   **一键安装 (`install`)**：脚本处理所有事情：备份现有配置、为 `~/.vim` 和 `~/.vimrc` 创建符号链接，以及从清单中克隆所有插件。

-   **自动更新 (`update`)**：通过自动从仓库拉取最新更改，保持所有插件处于最新状态。

-   **健康与状态检查 (`status`)**：提供了一种快速验证所有组件是否正确安装和链接的方法，并识别任何不属于官方清单的插件。

-   **安全清理 (`clean` 和 `uninstall`)**：提供安全的交互式命令来移除多余的插件，或者完全、干净地卸载整个配置，而不会触及源文件。

这种新的脚本化方法确保了配置的一致性、可靠性和速度，将这个个人 Vim 设置转变为一个真正可移植的开发环境。

## 详细变更

### 新增功能

- 添加了 `vim-manager` 脚本，用于管理 Vim 配置的完整生命周期
- 实现了插件清单系统，便于管理依赖
- 添加了自动备份功能，防止配置丢失
- 支持在多个设备间同步配置

### 改进

- 优化了插件的安装和更新过程
- 改进了错误处理和用户反馈
- 添加了详细的日志记录

### 修复

- 修复了插件安装时的权限问题
- 修复了符号链接处理中的边缘情况
- 改进了对非标准 Vim 安装路径的支持

## 升级说明

如果您是从旧版本升级，请按照以下步骤操作：

1. 备份您当前的 Vim 配置
2. 拉取最新代码
3. 运行 `./vim-manager install` 进行升级
4. 重启 Vim 以应用所有更改

## 已知问题

- 在某些终端模拟器中，图标可能无法正确显示，请确保使用支持 Nerd Fonts 的终端
- 首次安装时可能需要手动安装 Nerd Fonts 以获得最佳体验
