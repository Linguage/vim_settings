# 快捷键与使用指南

本文档提供了本 Vim 配置中所有自定义快捷键和重要插件命令的参考。

## 前置键 (Leader Key)

`Leader` 键是一个特殊的前缀键，用于触发自定义快捷键。在本配置中，`Leader` 键被映射为 **`空格`** 键。

所以，当你看到 `<leader>w` 这样的快捷键时，意味着你需要先按 `空格` 键，然后再按 `w` 键。

---

## 全局快捷键

以下快捷键在普通模式下可用，除非另有说明。

### 文件与会话

| 快捷键          | 功能                               |
| --------------- | ---------------------------------- |
| `<leader>w`     | 保存当前文件 (`:w!`)              |
| `<leader>q`     | 关闭当前窗口 (`:q`)                |
| `<leader>Q`     | 强制关闭所有窗口 (`:qa!`)          |
| `<leader>e`     | 编辑 `vimrc` 文件                  |
| `<leader>r`     | 重新加载 `vimrc` 文件              |
| `<leader>/`     | 清除搜索高亮                      |

### 窗口管理

| 快捷键          | 功能                               |
| --------------- | ---------------------------------- |
| `<C-h/j/k/l>`   | 切换到左/下/上/右侧的窗口         |
| `<leader>=`     | 均衡所有窗口大小                  |
| `<leader>+`     | 增加窗口高度（5行）               |
| `<leader>-`     | 减少窗口高度（5行）               |
| `<leader>>`     | 增加窗口宽度（5列）               |
| `<leader><`     | 减少窗口宽度（5列）               |

### 导航与编辑

| 快捷键          | 功能                               |
| --------------- | ---------------------------------- |
| `<leader>j`     | 向下移动10行                      |
| `<leader>k`     | 向上移动10行                      |
| `H`             | 移动到行首（非空白字符）           |
| `L`             | 移动到行尾                        |
| `<leader>d`     | 删除当前行                        |
| `<leader>a`     | 全选当前文件内容 (`ggVG`)          |
| `<leader>y`     | （可视模式）复制到系统剪贴板      |
| `<leader>p`     | 从系统剪贴板粘贴                  |

## 插件快捷键

### NERDTree 文件浏览器

| 快捷键          | 功能                               |
| --------------- | ---------------------------------- |
| `<leader>n`     | 切换 NERDTree 文件浏览器          |
| `<leader>f`     | 在 NERDTree 中定位并显示当前文件  |
| `<Enter>`       | （在 NERDTree 中）在新标签页打开选中的文件 |


### 插入模式快捷键

| 快捷键          | 功能                               |
| --------------- | ---------------------------------- |
| `Esc`           | 退出插入模式返回普通模式          |
| `<C-c>`         | 退出插入模式返回普通模式          |
| `jk` / `kj`     | 快速退出插入模式返回普通模式      |
| `<C-h/j/k/l>`   | 在插入模式下移动光标（左/下/上/右）|

## 自定义命令

本配置还提供了一些有用的自定义命令：

- `:Copilot setup` - 设置 GitHub Copilot 认证
- `:Copilot status` - 查看 Copilot 状态
- `:Copilot enable` - 启用 Copilot
- `:Copilot disable` - 禁用 Copilot
- `:Format` - 格式化当前文件（如果支持）
- `:W` - 以 root 权限保存文件
- `:Q` - 退出当前窗口
- `:Wq` 或 `:X` - 保存并退出

## 提示

1. GitHub Copilot 需要有效的 GitHub Copilot 订阅
2. 首次使用 Copilot 时需要运行 `:Copilot setup` 进行认证
3. 要查看所有可用的快捷键，可以查看 `mappings.vim` 文件
4. 使用 `:map` 命令可以查看当前所有映射
5. 要修改任何快捷键，请编辑 `settings/mappings.vim` 文件
